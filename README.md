🚨 PSAS — Predictive Safety Analytics System
📌 Overview

PSAS is an AI-powered mine safety simulation system that predicts risks, monitors conditions, and supports emergency decision-making in underground mining environments.

🎯 Problem
Mining hazards like gas leaks, low oxygen, and tunnel collapse can lead to fatal accidents, especially where systems lack real-time intelligence.

💡 Solution
PSAS provides a fully offline, lightweight system that:

Predicts worker risk using AI
Simulates real-time incidents
Tracks workers across zones
Supports rescue team decisions

⚙️ Features
🧠 AI Risk Prediction (ML-based)
📊 Live dashboard & forecasting
🗺 Worker tracking map
🚑 Rescue command center
🧾 Incident timeline
🤖 AI safety assistant (offline capable)
🖥️ Tech Stack

HTML • CSS • JavaScript • Plotly • Machine Learning Logic
